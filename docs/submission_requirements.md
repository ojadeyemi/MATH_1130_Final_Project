# Submission Requirements

What we need to submit for full marks (110 points total)

---

## 1. Final Report (max 3 pages)

**What to include:**

**Executive Summary (non-technical)**

- What question did we ask and why?
- What did we find? (precise, not overgeneralized)

**Technical Section**

- Data cleaning process - how we handled missing data, merged datasets, created features
- Analysis process - EDA steps, visualizations, hypothesis tests, patterns we found
- Methods used - what stats/tests we ran and why
- Assumptions and limitations

**Grading:**

- Insightfulness (20 marks)
- Written communication (10 marks)
- Wrangling/cleaning (20 marks)
- Analysis depth (50 marks)

---

## 2. Video Presentation (max 8 minutes)

**What to include:**

- Problem background - why this matters
- How we analyzed the data - main steps
- Key findings - what we discovered
- Use lots of visualizations, minimal technical jargon

**Grading:**

- Oral communication (10 marks)
- Contributes to insightfulness score (20 marks)

---

## 3. Code (Jupyter notebooks)

**What to submit:**

- 01_data_cleaning.ipynb
- 02_analysis.ipynb
- Any other notebooks (max 5 total)

**Requirements:**

- Clear comments on every code cell
- Markdown explaining what we're doing
- Label cells: cleaning, EDA, hypothesis testing, etc.
- Code must be runnable top-to-bottom

**Grading:**

- Part of analysis score (50 marks)
- Part of wrangling score (20 marks)

---

## Getting Full Marks - Checklist

**Analysis (50 marks):**

- [ ] Multi-step EDA with visualizations
- [ ] Test hypotheses and interpret results
- [ ] Explain why we chose each visualization/test
- [ ] Connect individual analyses to answer overall question
- [ ] Investigate multiple sub-questions
- [ ] Test statistical significance
- [ ] Check for interactions between variables
- [ ] Well organized, clear sections
- [ ] Discuss limitations

**Wrangling (20 marks):**

- [ ] Handle outliers properly
- [ ] Handle missing data (we have lots)
- [ ] Merge datasets correctly
- [ ] Create useful features (we made is_entry_level, Industry, etc.)
- [ ] Document everything in report

**Insightfulness (20 marks):**

- [ ] Answer a clear, interesting question
- [ ] Conclusions are precise and nuanced
- [ ] Everything backed up by analysis

**Communication (20 marks):**

- [ ] Report is clear and concise (≤3 pages)
- [ ] Video is clear and concise (≤8 min)

**Bonus points:**

- Creative additions (extra datasets, unique analysis)
- "X-Factor" stuff not covered above
